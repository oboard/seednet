# 交互式 TUI

不带参数运行 SeedNet 即可打开交互式终端界面：

```sh
seednet
```

## 界面概览

```
┌─────────────────────────────────────────────────────────┐
│  SeedNet                                                 │
├──────────────────────────────┬──────────────────────────┤
│  种子：[我的秘密网络        ] │  节点 (2)                │
│  [ 启动 ]                    │  10.0.1.2  直连  12ms    │
│                              │  10.0.1.3  中继  89ms    │
├──────────────────────────────┴──────────────────────────┤
│  日志                                                    │
│  [INFO] DHT 自举完成                                     │
│  [INFO] 新节点已连接: abc123                             │
└─────────────────────────────────────────────────────────┘
```

## 快捷键

| 按键 | 操作 |
|---|---|
| `Tab` | 在各面板间切换焦点 |
| `Enter` | 启动 / 停止网络 |
| `↑` / `↓` | 滚动节点列表或日志面板 |
| `Ctrl+C` 或 `q` | 退出 |

## 操作流程

1. 在**种子**输入框中输入密码短语
2. 按 **Enter** 或点击**启动**
3. 观察**节点**面板——已连接的设备会自动出现
4. 查看**日志**面板了解连接事件
5. 再次按 **Enter**（或点击**停止**）断开连接

## 提示

- 种子输入框随时可编辑——启动前可随时修改
- 日志面板实时追踪 `~/.seednet/seednet.log`
- 节点列表中显示的覆盖 IP 就是你与对端通信时使用的地址（如 `ping 10.0.1.2`）
