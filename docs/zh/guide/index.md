# SeedNet 是什么？

SeedNet 是一个**去中心化私有覆盖网络**。它让你只需一个共享的密码短语——你的*种子*——就能将任意数量的设备连接成同一个虚拟局域网。

## 一句话说清楚

每台运行 `seednet up "我的秘密短语"` 的设备，只要短语相同，就会通过 BitTorrent DHT 网络自动互相发现，并建立加密的点对点隧道——无需账户、无需云控制器、无需端口转发。

## 核心概念

| 概念 | 含义 |
|---|---|
| **种子短语** | 唯一的共享密钥。使用相同短语的任何设备都会加入同一网络。请妥善保管。 |
| **覆盖 IP** | 每台设备都会获得一个稳定的虚拟 IP（如 `10.x.x.x`），由其身份派生。像局域网地址一样使用。 |
| **身份** | 存储在 `~/.seednet/identity.bin` 的每设备 Ed25519 密钥对。不由种子派生——每台设备唯一。 |

## 有哪些用途？

- 从任何地方访问家里的服务器
- 在跨网络的机器之间建立安全链路
- 联机游戏、文件共享、远程桌面——无需暴露端口
